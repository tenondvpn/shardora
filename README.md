# Shardora
      A Dynamic and Public Blockchain Sharding Platform with Seamless Shards Reconfiguration

# Quick Start
## Requirements
      centos7
      g++8.3.0
      python3.10+

## Run local shardora network
      git clone git@github.com:tenondvpn/shardora.git
      sh local.sh

## Transaction test
      cd ./cbuild_Debug && make txcli
      ./txcli
